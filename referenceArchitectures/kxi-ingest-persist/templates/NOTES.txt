Installed kdb Insights Ingest and Persist - RT Based Reference Architecture
================================

You have installed an instance of the kdb Insights Ingest and Persist - RT Based, Reference Architecture chart. This deploys the following components using their independent charts:
- A kxi-db chart encompassing the core elements of the InsightsDB which can ingest and persist data
  See docs for more info: https://code.kx.com/insights/microservices/database/index.html
- A kxi-rt chart as the message bus to log the ingested data and publish to the kxi-db chart
  See docs for more info: https://code.kx.com/insights/microservices/rt/index.html
- A kxi-gw chart used to query the data from the kxi-db chart
  See docs for more info: https://code.kx.com/insights/microservices/database/query/index.html
